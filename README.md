# github_wiki_Dio
Projeto: desenvolvendo uma Wiki de Repositórios do GitHub Com React

O projeto consiste em uma aplicação em React que utiliza a API do Github para recuperar dados de um repositório qualquer e os adicionaem uma lista para ser visualizada na tela.
